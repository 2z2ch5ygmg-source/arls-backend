# ARLS Employee Drawer Step 1 Manual Checklist

- 직원 목록에서 행을 클릭하면 우측 드로어가 열린다.
- 드로어 폭이 기존보다 넓고, 전체가 프로필 패널처럼 보인다.
- 헤더에서 이름/직원번호/역할/재직상태를 즉시 읽을 수 있다.
- 헤더에 회사, 지점, 입사일, 계정 연동 상태가 보인다.
- 빠른 액션이 헤더 근처에 보인다.
- 탭 수가 4개인지 확인한다.
- 기본 진입 탭이 `개요`인지 확인한다.
- `개요` 탭에 KPI + 근무 정보 + 최근 활동이 보이는지 확인한다.
- 출퇴근 데이터가 없는 직원은 여러 0카드 대신 빈 상태 블록으로 보이는지 확인한다.
- 스케줄 데이터가 없는 직원은 빈 상태 블록으로 보이는지 확인한다.
- 휴가·요청 데이터가 없는 직원은 `휴가·요청` 탭에서 압축된 empty state로 보이는지 확인한다.
- `직원 수정`, `스케줄 보기`, `출퇴근 보기`, `휴가 보기`, `요청 보기` 버튼이 실제 라우트 이동을 여는지 확인한다.
- 드로어 닫기 버튼이 정상 동작하는지 확인한다.
- 다른 직원을 클릭하면 헤더와 내용이 즉시 바뀌는지 확인한다.

# ARLS Functional Restore Pass 2 Manual Checklist

- `스케쥴 > 근무일정 > Excel로 근무표 간편 제작`에서 STEP 4가 보이는지 확인
- STEP 3에서 source artifact가 없는 지점/월이면 STEP 4 업로드 파일과 검토 버튼이 잠기는지 확인
- STEP 3에서 source artifact가 준비된 지점/월이면 STEP 4에서 파일 선택이 가능한지 확인
- HQ 작성본 workbook을 선택하면 `검토 시작`이 활성화되는지 확인
- `검토 시작` 후 review summary, issue groups, row review table이 ARLS 안에 표시되는지 확인
- 차단 이슈가 있으면 `ARLS에서 적용`이 비활성화되거나 apply blocked 결과가 보이는지 확인
- 차단 이슈가 없으면 `ARLS에서 적용` 후 적용 결과 수치가 ARLS 화면에 남는지 확인
- 적용 후 `ticket 갱신`, `자동승인`, `승인대기`, `알림`, `ARLS bridge` 수치가 표시되는지 확인
- site 또는 month를 바꾸면 기존 STEP 4 검토 결과가 stale 상태로 무효화되는지 확인
- 보고 탭은 shortcut만 보여주고, 실제 upload/review/apply는 Excel workflow STEP 4에서 진행되는지 확인

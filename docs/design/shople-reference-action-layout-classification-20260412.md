## Reference Traits To Preserve

- Use Shople as visual and interaction reference, not feature-copy target.
- Keep the work surface light, table/list-first where operational comparison is the job.
- Use compact top toolbars and domain-local actions instead of launcher cards.
- Keep sidebar navigation as navigation; content should own the work action, not repeat the entire nav.
- Prefer list + detail, calendar + detail, or wizard + active section patterns over card walls.
- Do not reintroduce heavy box-in-box, decorative title icons, U-shaped tabs, or oversized empty-state cards.

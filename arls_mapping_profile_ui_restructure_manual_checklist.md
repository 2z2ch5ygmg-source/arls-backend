# ARLS Mapping Profile UI Restructure Manual Checklist

- `근무 템플릿` 화면에 `근무 템플릿 / 매핑 프로필` 내부 탭이 보이는지 확인
- `근무 템플릿` 탭의 템플릿 행마다 `수정 / 복제 / 비활성화 / 삭제`가 보이는지 확인
- 사용 중인 템플릿의 `삭제` 버튼이 비활성화되고 사유가 표시되는지 확인
- `매핑 프로필` 탭에서 현재 업로드 프로필 row와 규칙 수/상태/수정일이 보이는지 확인
- `매핑 프로필` 탭의 `프로필 편집` 버튼이 editor를 여는지 확인
- `Excel로 근무표 간편 제작` Step 1에 `매핑 프로필 선택` selector가 보이는지 확인
- Step 1에서 선택된 프로필의 프로필명/범위/상태/수정일이 보이는지 확인
- Step 1에서 규칙 요약이 `주간 10시간 -> 템플릿 A` 형식으로 보이는지 확인
- profile 미선택 또는 비활성/누락 상태면 `다음` 버튼이 비활성화되는지 확인
- Step 1의 `매핑 프로필 관리` 버튼이 `근무 템플릿 > 매핑 프로필` 탭으로 이동시키는지 확인
